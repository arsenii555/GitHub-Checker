Welcome to GitHub Checker's documentation!
============================================

.. image:: _static/GitHub_Checker_logo.png

GitHub Checker - это система для отслеживания изменений в репозиториях GitHub и уведомления пользователей через Telegram бота.

Основные возможности
--------------------

- Отслеживание изменений в репозиториях GitHub
- Уведомление пользователей в Telegram при изменениях
- Управление подписками через Telegram бота:

  - Добавление репозиториев для отслеживания (``/track <ссылка>``)

  - Просмотр списка отслеживаемых репозиториев (``/list``)
  - Удаление репозиториев из списка (``/untrack <ссылка>``)

Архитектура
-----------

Проект состоит из двух основных компонентов:

1. **Telegram бот** - взаимодействует с пользователями, обрабатывает команды
2. **Scrapper** - периодически проверяет репозитории на изменения и отправляет уведомления
